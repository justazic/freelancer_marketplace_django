# 💼 Freelance Marketplace (Django MVT Version)

Ushbu platforma freelancerlar va mijozlar o'rtasida ish jarayonini avtomatlashtirish uchun mo'ljallangan. Mijozlar loyihalar joylaydi, freelancerlar o'z takliflarini bildirishadi va shartnoma asosida hamkorlik qilishadi.

---

## 🛠 Texnologik Stek
* **Backend:** Python 3.10+, Django 5.x
* **Ma'lumotlar bazasi:** PostgreSQL 
* **Frontend:** Django Templates, CSS3 (Custom Dashboard UI), FontAwesome Icons
* **Arxitektura:** Model-View-Template (MVT)

## 🌟 Asosiy Imkoniyatlar

### 🏢 Mijozlar (Clients) uchun:
* **Loyihalar boshqaruvi:** Loyiha yaratish (title, description, budget, deadline).
* **Takliflarni ko'rish:** Freelancerlardan kelgan barcha "Bid"larni ko'rish va saralash.
* **Freelancer tanlash:** Eng mos taklifni qabul qilish (Contract avtomatik yaratiladi).
* **Baholash tizimi:** Loyiha tugagach, freelancerga 1-5 gacha reyting va sharh qoldirish.

### 👨‍💻 Freelancerlar uchun:
* **Loyiha qidirish:** Barcha ochiq loyihalarni (Open projects) ko'rish.
* **Taklif yuborish:** Loyihalarga o'z narxi va xabari bilan "Bid" yuborish.
* **Dashboard:** Aktiv shartnomalar, yuborilgan takliflar va o'rtacha reytingni kuzatish.
* **Chat tizimi:** Shartnoma doirasida mijoz bilan muloqot qilish.
